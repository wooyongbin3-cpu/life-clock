## 行为摘要

### 每帧（随屏幕刷新，可达 ~120Hz）

- 年龄秒：`SS.mmm`（基于 `totalMs`，非墙钟秒）
- 今日 / 今年 / 人生进度环 `stroke-dashoffset`
- 百分比文案（2 位小数，脏检查）

### 每秒 / 心跳 / 分钟变化

- 年/月/日/时/分整数格
- 累计时长、里程碑、章节、诗句轮播

### 性能点

- 去掉 `.life-ring .progress { transition: 0.6s }`（原先会拖 0.6s）
- `setMetric` / 文本写入前做相等判断，减少 120fps 下 DOM 抖动
- 预计算 `RING_C`，避免每帧 `2 * Math.PI * 40`
