# Life Clock 发布说明

## 🌟 功能特性

生命时钟是一款温柔而深情的时间应用。它不会催促任何人，只是温柔地帮助你看见时间的流逝。

### ✨ 核心功能
- **精确计时**：从出生至今的年月日时分秒
- **人生里程碑**：10,000 天、1 亿秒、30 岁、50 岁等
- **生命章节**：童年 → 少年 → 青春期 → 而立 → 不惑 → 花甲 → 期颐
- **进度可视化**：今日进度、今年进度、至 80 岁的生命进度环
- **诗句轮播**：每 12 秒更新一条诗句，温柔陪伴
- **心跳计数**：按自定义心率估算总跳动次数
- **三套主题**：暖光、暮色、墨蓝，适应不同时刻

### 🔒 隐私保护
- ✅ **完全本地存储** - 数据仅保存在你的设备上
- ✅ **无后端服务** - 无云同步，无服务器
- ✅ **无用户追踪** - 无分析脚本，无广告
- ✅ **离线可用** - 无网络也能使用

## 📥 安装方式

### 方式 1: 直接安装（推荐）
1. 下载本文件下方的 `app-debug.apk`
2. 在 Android 手机上打开下载的文件
3. 按提示完成安装
4. 完成！

> **注意**：如果提示"未知来源应用"，请在设置中允许。

### 方式 2: 命令行安装（开发者）
```bash
adb install app-debug.apk
```

## 🎯 快速开始

1. **打开应用** → 输入你的出生年月日
2. **可选**：设置昵称、心率、显示偏好
3. **查看**：你的人生时间轴、里程碑、进度
4. **享受**：让数字安静地陪伴你
